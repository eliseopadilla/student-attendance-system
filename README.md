# Student Attendance Management System
Backend-driven student attendance system with relational database design (Node.js + MySQL).

Web system to register daily student attendance (manual selection) and generate reports for teachers.

## Tech Stack
- Node.js
- JavaScript, HTML, CSS
- MySQL
- Postman

## Files
- `Asistencias.sql`: database schema and initial setup
- Frontend: `*Index.html`, `Style2.css`, `script*.js`
- Reports: `reportes.html`, `reportes.js`
